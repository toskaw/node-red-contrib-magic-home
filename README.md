Node-RED nodes for integration MagicHome bulb
I use HaoDeng WIFI SMART BULB RGBCW
https://www.amazon.co.jp/gp/product/B07XSCY5GN

Nodes:
* on
* rgb
* cw
* status
  
Installation:
```
npm install node-red-contrib-magic-home-tos
```
